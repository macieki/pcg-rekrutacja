# University Recruitment System Prototype

A clickable recruitment system prototype built with Vue 2, Nuxt 2, and Vuetify 2.

## Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# build for production and launch server
npm run build
npm run start

# generate static project
npm run generate
```

## Features

- Multi-step registration and application process
- Programme catalogue with filtering
- Personal data and education wizards
- Dashboard with application tracking
- Mock authentication and data storage
